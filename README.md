## :dart: About the project

forkify is a modern recipe app made as part of Jonas Schmedtmann JS Course (using MVC pattern)

## :sparkles: Features

:heavy_check_mark: Searching for recipes\
:heavy_check_mark: Adding own recipes\
:heavy_check_mark: Bookmarking recipes, using local storage\
:heavy_check_mark: Contact form

## :rocket: Technologies

Tools used in this project:

- Vanilla JS
- Sass
- Parcel
